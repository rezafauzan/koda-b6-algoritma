# Algorithma untuk menentukan apakah suatu bilangan merupakan bilangan ganjil atau genap

1. Mulai
2. Siapkan sebuah nilai bilangan
3. Cek apakah bilangan tersebut habis jika dibagi dengan 3 jika ya maka bilangan ini adalah ganjil
4. Jika tidak cek apakah bilangan tersebut habis dibagi 2 jika ya maka bilangan ini genap
5. jika tidak juga berarti ini bukan sebuah bilangan
6. Selesai
